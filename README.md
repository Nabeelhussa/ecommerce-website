# 🛒 E-Commerce Website Project

## 📌 Project Overview

This project is a fully responsive **E-commerce website** built using HTML, CSS, and JavaScript.
It follows a structured weekly development process based on a Figma design.

The website includes product browsing, detailed product pages, and a shopping cart system.

##  Features

*  **Homepage**

  * Hero banner with call-to-action
  * Featured categories and deals section

*  **Product Listing**

  * Grid and list view options
  * Filtering sidebar (brand, price, ratings)
  * Hover effects for better UX

*  **Product Details Page**

  * Image gallery with thumbnails
  * Product description and pricing
  * Related products section

*  **Shopping Cart**

  * Add/remove products
  * Quantity update
  * Order summary with total calculation

*  **Reusable Components**

  * Header and Footer managed via JavaScript (`components.js`)
  * Consistent styling using global CSS (`styles.css`)
